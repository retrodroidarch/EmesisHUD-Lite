# EmesisHUD Lite

EmesisHUD Lite is a custom Team Fortress 2 HUD designed for speed, clarity, and tactical play.  
This release focuses on the essentials: killfeed, scoreboard, health, ammo, crosshair, and medic charge.

## Features
- Custom Killfeed (HudDeathNotice)
- Redesigned Scoreboard
- Clean Health + Ammo panels
- Streamlined Crosshair + Medic Charge
- Modular design for future expansion

## Installation
1. Download the latest release from this repository.
2. Extract the folder, Put it into `tf/custom/`.
3. Launch TF2, You will see the differences. No console command needed.
